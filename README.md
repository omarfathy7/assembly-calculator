# Assembly Calculator

A simple calculator program written in x86 Assembly language (MASM).

## Features
- Addition
- Subtraction
- Multiplication
- Division (with zero-check)
## Operations
- Press `1` for Addition
- Press `2` for Subtraction
- Press `3` for Multiplication
- Press `4` for Division
- Press `ESC` to Exit
